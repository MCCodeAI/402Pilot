Agent Planner task descriptor type · prompt · budget snapshot
402Pilot decision layer
1 · Context Encoder x_t task type + difficulty + wallet + provider EWMA
2 · Budget Manager λ_norm = λ / (1 + λ) affordability filter + budget pressure
3 · Service Selector a* = argmax [(1-λ)·û_k - λ·ĉ_k] PA-DCT: contextual TS + quality/cost posteriors
6 · Policy Updater posterior(q, c) ← utility, cost discount γ keeps the learner adaptive
x402 Executor pay_and_call(a*) settlement is unchanged
Provider Market P-cheap · P-mid · P-premium · P-adv · P-flaky cost · latency · response · failure
Evaluator q ∈ [0, 1] pass@1 · EM/F1 · cached judge score
Reward Calculator utility = q - ν·failure PA_reward = (1-λ)·utility - λ·c̃ posterior updates use utility; accounting uses PA_reward
选择路径task → x_t → λ_norm → a* → x402 payment → provider
学习路径response/cost/failure → q → utility / PA_reward → posterior(q,c) → next round
